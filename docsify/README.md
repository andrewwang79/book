# 使用
1. 安装本地工具，依赖Node.js：npm i docsify-cli -g
1. 创建空项目：docsify init XYZ
1. 启动服务：cd XYZ && docsify serve
1. 使用Github Pages部署文档，推荐用master：https://juejin.im/post/5d6889d851882518980de205
